# תכנות מערכות א' - מטלה 1: ספריות ומיון מספרים

פרויקט זה בוצע במסגרת קורס תכנות מערכות א'. המטרה המרכזית של המטלה היא מימוש פונקציות לסיווג מספרים, עבודה עם ספריות סטטיות ודינמיות, וניהול תהליך הבנייה באמצעות `Makefile`.

## תיאור הפרויקט
התוכנית מקבלת מהמשתמש שני מספרים שלמים חיוביים (טווח) ומדפיסה את כל המספרים בטווח זה שעונים על הקטגוריות הבאות:
* **מספרים ראשוניים (Primes):** מספרים שמתחלקים רק ב-1 ובעצמם.
* **מספרי פלינדרום (Palindromes):** מספרים שנקראים אותו דבר מההתחלה לסוף ולהיפך.
* **מספרי ארמסטרונג (Armstrong numbers):** מספר שסכום חזקות ספרותיו (לפי כמות הספרות) שווה למספר עצמו.
* **מספרים חזקים (Strong numbers):** מספר שסכום העצרת של ספרותיו שווה למספר עצמו (לדוגמה: $1! + 4! + 5! = 145$).

## מבנה הספריות
הפרויקט מדגים שימוש בשתי גישות מימוש שונות עבור פונקציות ה-Advanced:
1. **גרסת הלולאות (Loops):** מימוש באמצעות לולאות `while` ו-`for`.
2. **גרסת הרקורסיה (Recursion):** מימוש באמצעות פונקציות רקורסיביות.

## קבצי הפרויקט
* ה-`NumClass.h`: קובץ ה-Header המכיל את הגדרות הפונקציות.
* ה-`basicClassification.c`: מכיל את הפונקציות `isPrime` ו-`isStrong`.
* ה-`advancedClassificationLoop.c`: מכיל את `isArmstrong` ו-`isPalindrome` במימוש לולאות.
* ה-`advancedClassificationRecursion.c`: מכיל את `isArmstrong` ו-`isPalindrome` במימוש רקורסיבי.
* ה-`main.c`: התוכנית הראשית המנהלת את קלט המשתמש והדפסת התוצאות.
* ה-`makefile`: לניהול הקומפילציה ויצירת הספריות.

## פקודות קומפילציה והרצה
ניתן לבנות את הפרויקט במגוון דרכים באמצעות ה-`Makefile`:

### יצירת ספריות
* עבור `make loops` - יצירת ספרייה סטטית של גרסת הלולאות (`libclassloops.a`).
* עבור `make recursives` - יצירת ספרייה סטטית של גרסת הרקורסיה (`libclassrec.a`).
* עבור `make loopd` - יצירת ספרייה דינמית של גרסת הלולאות (`libclassloops.so`).
* עבור `make recursived` - יצירת ספרייה דינמית של גרסת הרקורסיה (`libclassrec.so`).

### יצירת קבצי הרצה
* פקודת `make mains` - יוצרת קובץ הרצה המקושר לספרייה הסטטית הרקורסיבית.
* פקודת`make maindloop` - יוצרת קובץ הרצה המקושר לספרייה הדינמית של הלולאות.
* פקודת `make maindrec` - יוצרת קובץ הרצה המקושר לספרייה הדינמית הרקורסיבית.

### ניקוי
*פקודת `make clean` - מוחקת את כל קבצי ה-`.o`, הספריות וקבצי ההרצה.
